CNZ

Call if not zero


If Zero = 0 subroutine will be called.

Instruction Hex code Length in bytes Length in cycles
CNZ a16 0xC4 3 17/11

*a16 means 16 bits of address.

*length depends on action.

Example

MVI
B,05
MVI
A,10
SUB
B
CNZ
label
HLT

Because 0x10 - 0x05 does not equal 0 flag Zero will be set to 0 and subroutine will be called.
