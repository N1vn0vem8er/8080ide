CNC

Call if not carry


Jeżeli flaga Carry = 0 program przechodzi do podprogramu.

Instrukcja Kod hex Długość w bajtach Długość w cyklach
CNC a16 0xD4 3 17/11

*a16 oznacza 16 bitów adresu.

*długość zależy od wykonania lub nie wykonania akcji.

Przykład

MVI
B,05
MVI
A,10
SUB
B
CNC
label
HLT

Ponieważ przy 0x10 - 0x05 nie występuje przeniesienie podprogram nie zostanie wywołana.
