CMP

Compare register or memory with accumulator


Compares the contents of the accumulator with the register or memory. Depending on the result, sets the appropriate bits of the flag register. If the Zero bit is set to 1, it means that the values ​​are equal, if to 0, they are not equal. If Carry is set to 1, the value in the register is greater than the value in the accumulator, if Carry is set to 0, the value in the accumulator is greater.

Condition bits affected: Carry, Zero, Sign, Parity, Auxiliary Carry.

Instruction Hex code Length in bytes Length in cycles
CMP B 0xB8 1 4
CMP C 0xB9 1 4
CMP D 0xBA 1 4
CMP E 0xBB 1 4
CMP H 0xBC 1 4
CMP L 0xBD 1 4
CMP M 0xBE 1 7
CMP A 0xBF 1 4

Example

MVI
A,F2
MVI
B,04
CMP
B

After executing these Instruction Carry will be set to 0 which means contents of accumulator are greater then contents of register B.
