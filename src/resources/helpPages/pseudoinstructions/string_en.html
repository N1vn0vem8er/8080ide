STRING


Pseudo instruction used to save chain of characters into memory. Characters must be between "".

Example

MOV
A,B
STRING
"hello word"

Chain of characters will be saved after instruction MOV A,B.

If code of MOV A,B is located under address 0x1001 the memory will look like this:
