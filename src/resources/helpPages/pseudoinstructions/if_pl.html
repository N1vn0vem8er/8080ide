IF


Pseudo instrukcja sprawdzająca wartość stałej i w zależności od jej wartość umieszcza lub pomija instrukcje do pseudoinstrukcji ENDIF. Jeżeli stała ma wartość 0 to instrukcje nie są asemblowane, jeżeli stała ma jaką kolwiek inną wartość to są. UWAGA Nameży używać z pseudo instrukcją ENDIF.

Przykład

const: EQU 0
IF
const
MOV
A,B
MVI
A,65
ENDIF

Stała "const" jest ustawiana na 0 przez co instrukcje pomiędzy IF i ENDIF nie znajdą się w wynikowym kodzie.
