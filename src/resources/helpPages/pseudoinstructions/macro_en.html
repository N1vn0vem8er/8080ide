MACRO


Pseudo instruction creating macro. The following pseudo instructions cannot be used inside macro: IF, ENDIF, EQU, SET and definition of other macros. Warning always use with ENDM.

Example

name: MACRO
MOV
A,B
MVI
A,65
ENDM

Macro named "name" will be created. It will contain instructions: MOV A,B i MVI A,65.

RRC
A,B
name
MOV
B,C
MVI
C,60

In the place of "name" macro will be inserted.
