STRING


Pseudo instrukcja służąca do zapisania ciągu znaków w pamięci pod adresem ustawienia pseudo instrukcji. Ciąg znaków musi być zapisany pomiędzy "".

Przykład

MOV
A,B
STRING
"hello word"

Ciąg znaków zostanie zapisany w pamięci po instrukcji MOV A,B.

Zakładając, że kod instrukcji MOV A,B znajduję się w pamięci pod adresem 0x1001 pamięć będzie wyglądać następująco:
Adres Pamięć
0x1001 0x78
0x1002 0x68
0x1003 0x65
0x1004 0x6c
0x1005 0x6c
0x1006 0x6f
0x1007 0x20
0x1008 0x77
0x1009 0x6f
0x100A 0x72
0x100B 0x64
