SET


Pseudo instrukcja ustawiająca wartość do symbolu. Wartość symbolu może być ustwaiona wiele razy. UWAGA Należy ustawić przed jej użyciem.

Przykład

name: SET 1002
JMP
name
name: SET 0B32
JMP
name

Symbol "name" jest najpierw ustawiany na 0x1002 i pierwsza instrukcja JMP będzie skakała do tej wartości, następnie symbol "name" jest ustawiany na 0x0B32 przez co druga instrukcja JMP będzie skakać do adresu 0x0B32.
JMP
1002
JMP
0B32
