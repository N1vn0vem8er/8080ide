ARRAY


Pseudo instruction saving data as array. Data can only be written in hexadecimal number system.

Example

MOV
A,B
ARRAY
1234ABC54

Data in pseudo instruction ARRAY will be saved after instruction MOV.

If code of instruction MOV A,B is located in address 0x1001 memory will look like this:
Address Memory
0x1001 0x78
0x1002 0x12
0x1003 0x34
0x1004 0xAB
0x1005 0xC5
0x1006 0x04
Last digit 4 is saved as 0x04. If you want to save it as 0x54 you should put 0 in front like below.

ARRAY
01234ABC54
