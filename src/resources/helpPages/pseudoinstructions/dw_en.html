DW


8 least significant bits of data are stored in memory, 8 most significant bits are stored under next memory address.

Example

MOV
A,B
ldata: DW 954F

Data will be stored after code of
MOV
.

If code of instruction MOV A,B is located in memory under address 0x1001 memory will look like this:
Address Memory
0x1001 0x78
0x1002 0x4F
0x1003 0x95
