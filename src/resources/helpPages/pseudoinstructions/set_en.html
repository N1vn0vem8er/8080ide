SET


Pseudo instruction seting value to symbol. Value can be set multiple times. WARNING Set before using.

Example

name: SET 1002
JMP
name
name: SET 0B32
JMP
name

"name" will be set to 0x1002 and first JMP will jump to this value, next "name" is set to 0x0B32 and second JMP will jump to address 0x0B32.
JMP
1002
JMP
0B32
