MACRO


Pseudo instrukcja tworząca makro. Wewnątrz makra nie można umieszczać pseudo instrukcji IF, ENDIF, EQU, SET i definicji innych makr. UWAGA Nameży używać z pseudo instrukcją ENDM.

Przykład

name: MACRO
MOV
A,B
MVI
A,65
ENDM

Makro o nazwie "name" zostanie utworzone i będzie można go używać w kodzie. Będzie on zawierać instrukcje MOV A,B i MVI A,65.

RRC
A,B
name
MOV
B,C
MVI
C,60

W miejscu "name" wstawione zostaną instrukcje zapisane w makrze.
