IF


Pseudo instruction checks constant and depending on its value assembler assembles of not instructions to pseudo instruction ENDIF. If constant value is H0000 instrutcions are not assembled otherwise they are. WARNING Must be used with ENDIF.

Example

const: EQU 0
IF
const
MOV
A,B
MVI
A,65
ENDIF

Constant "const" is set to 0x0 and instructions between IF and ENDIF will not be assembled.
