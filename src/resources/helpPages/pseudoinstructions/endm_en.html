ENDM


This pseudo instruction marks the end of macro. WARNING always use with MACRO.

Example

name: MACRO
MOV
A,B
MVI
A,65
ENDM

Macro named "name" will be created and and you can use it in code. This macro will contain MOV A,B and MVI A,65.

A,B
name

B,C

C,60
