STA

Store accumulator direct


Saves contents of register A under memory address.

Instruction Hex code Length in bytes Length in cycles
STA a16 0x02 3 13

*a16 means 16 bits of address.

Example

MVI
A,AB
STA
0012

After executing these instructions contents of register A will be saved under memory address 0x0012.

Before STA Address After STA
0x00 0x0012 0xAB
