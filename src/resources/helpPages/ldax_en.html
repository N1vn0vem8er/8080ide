LDAX

Load accumulator


Loads to accumulator data from memory location addresses by registers B and C or D, and E.

Instruction Hex code Length in bytes Length in cycles Description
LDAX B 0x0A 1 7 Address from registers B and C.
LDAX D 0x1A 1 7 Address from registers D and E.

Example

MVI
B,00
MVI
C,10
LDAX
B

After executing these instruction accumulator will contain data from memory address 0x0010.
