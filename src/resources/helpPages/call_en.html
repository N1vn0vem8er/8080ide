CALL

Call


Unconditionally perform subroutine. Return address is pushed onto the stack and program jumps to address of subroutine. To return from subroutine use RET.

Instruction Hex code Lenght in bytes Lenght in cycles
CALL a16 0xCD 3 17

*a16 means 16 bits of address.

Example

MVI
B,05
MVI
A,10
SUB
B
CALL
label
HLT

Subroutine will always be performed.
