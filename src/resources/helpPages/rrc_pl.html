RRC

Rotate accumulator right


Zawartość akumulatora jest przesuwana o jedną pozycje w prawo. Najbardziej znaczący bit zapisywany jest w fladze Carry

Instrukcja Kod hex Długość w bajtach Długość w cyklach
RRC 0x0F 1 4

Przykład

MVI
A,F2
RRC

Po wykonaniu tych instrukcji zawartośc rejestru A będzie 0x79 i flaga Carry zostanie ustawiona na 0.

Przed wykonaniem

Carry = x
1 1 1 1 0 0 1 0

Po wykonaniu

0 1 1 1 1 0 0 1
Carry = 0
