CPO

Call if parity odd


If Parity = 0 subroutine is called.

Instruction Hex code Length in bytes Length in cycles
CPO a16 0xE4 3 17/11

*a16 means 16 bits of address.

*length depends on action.

Example

MVI
B,05
MVI
A,10
SUB
B
CPO
label
HLT

Because 0x10 - 0x05 = 0x0B which is not even and Parity is set to 0 so subroutine is called.
