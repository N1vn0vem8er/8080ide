LHLD

Load H and L direct


Loads registers H and L with 16-bits of data saved at memory location addresed by 16-bits from after instruction code.

Instruction Hex code Length in bytes Length in cycles
LHLD a16 0x2A 3 16

*a16 means 16 bits of address.

Example

LHLD
0013

If memory will look like below register H will be loaded with 0xA6, and register L with 0x32.

Address Memory
0x0013 0x32
0x0014 0xA6
