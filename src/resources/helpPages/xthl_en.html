XTHL

Exchange Stack


Contents of register L are exchanged with data pointed by stack pointer SP. Contents of register H are exchanged with data pointed by SP + 1.

Instruction Hex code Length in bytes Length in cycles
XTHL 0xE3 1 18

Example

XTHL

If SP contains 0x10AD, register H 0x0B and register L 0x3C. Memory looks like below. After executing this instruction registers H and L will contain 0x0D, and 0xF0.

Memory before Address Memory after
