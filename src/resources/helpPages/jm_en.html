JM

Jump if minus


If Sign = 1 execution will continue on specified memory address.

Instruction Hex code Length in bytes Length in cycles
JM a16 0xFA 3 10

*a16 means 16 bits of data.

Example

MVI
B,H3F
label: INR B
JM
label
HLT

Becouse register B after incrementation will be 0x40 Sign = 0, so execution will not contiune on address of "label".
