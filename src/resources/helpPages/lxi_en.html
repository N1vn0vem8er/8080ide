LXI

Load register pair immediate


Two bytes of immediate data are loaded into pair of registers or stack pointer.

Instruction Hex code Length in bytes Length in cycles Description
LXI B,d16 0x06 3 10 Data is loaded into B and C.
LXI D,d16 0x16 3 10 Data is loaded into D and E.
LXI H,d16 0x26 3 10 Data is loaded into H and L.
LXI SP,d16 0x36 3 10 Data is loaded into SP.

*d16 means 16 bits of data.

Example

LXI
B,1243

After executing these instructions register B will contain 0x12 and register C 0x43.

Example instruction in memory:

Address Memory
0x0045 0x01
0x0046 0x12
0x0047 0x43
