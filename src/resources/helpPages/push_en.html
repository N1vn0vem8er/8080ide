PUSH

Push data onto stack


The instruction places the contents of the specified register pair in memory at the address pointed to by the SP stack pointer. The contents of the first register are written at an address one less than the address stored in the SP, and the contents of the second register are written at an address two less than the contents of the SP. When PSW is used, the first byte saves the contents of the A register, the second one stores the contents of the flag register.

Instruction Hex code Length in bytes Length in cycles Description
PUSH B 0xC5 1 10 Data from register pair B and C is loaded on stack.
PUSH D 0xD5 1 10 Data from register pair D and E is loaded on stack.
PUSH H 0xE5 1 10 Data from register pair H and L is loaded on stack.
PUSH PSW 0xF5 1 10 Data from flags register and accumulator is loaded on stack.

Example

MVI
C,F2
MVI
B,52
PUSH
B

If SP contains 0x1245 after executing these instructions under memory addresses 0x1244 and 0x1243 0x52, and 0xF2 will be saved. SP will be 0x1243.
