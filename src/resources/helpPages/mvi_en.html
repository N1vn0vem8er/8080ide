MVI

Move immediate data


One byte of immediate data is loaded to register or memory.

Instruction Hex code Length in bytes Length in cycles Description
MVI B,d8 0x06 2 7 Data is loaded to register B.
MVI D,d8 0x16 2 7 Data is loaded to register D.
MVI H,d8 0x26 2 7 Data is loaded to register H.
MVI M,d8 0x36 2 10 Data is loaded to register M.
MVI C,d8 0x0E 2 7 Data is loaded to register C.
MVI E,d8 0x1E 2 7 Data is loaded to register E.
MVI L,d8 0x2E 2 7 Data is loaded to register L.
MVI A,d8 0x3E 2 7 Data is loaded to register A.

*d16 means 16 bits of data.

Example

MVI
A,10

After executing this instruction register A will contain 0x10.

Example instruction in memory.

Address Memory
0x0045 0x3E
0x0046 0x10
