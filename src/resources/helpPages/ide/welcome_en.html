Welcome in help center


About help center

The Help Center is a place where you can find help with using the program, assembler instructions and pseudo-instructions.

Using help center

You can open the Help Center using the button in the Help -> Help Center menu or by pressing the F1 key. When you have a code file open and want help with an instruction, place the cursor on the line with the instruction and press F1. Help will open on the page regarding the instructions in the line that was indicated. You can customize where the help opens in the settings found in the Edit -> Options -> Help Settings menu.

The help categories are located at the top of the window. Clicking on them will take you to a given category. On the left side there is a page selection menu. Clicking on one of the options displayed there will open the help page for that option.
