Projekty


Projekty umożliwiają tworzenie programów z wielu plików, używanie integracji z systemem kontroli wersji git jak również ustawianie symulatora. Są one preferowanym sposobem na tworzenie średnich i większych programów.

Tworzenie projektu

Aby stworzyć projekt naciśnij menu Projekt -> Nowy Projekt. Pojawi się okno proszące o podanie miejsca utworzenia projektu, jego nazwę i początkowo ustawioną wielkość pamięci symulatora. Po naciśnięciu przycisku Ok projekt zostanie utworzony. Zainicjowane zostanie też repozytorium Git. Jeżeli w konfiguracji projektu są jakieś błędy nie zostanie on utworzony.

Konfiguracja projektu

Projekt konfiguruje się poprzez edytowanie pliku {nazwa projektu}.json lub poprzez naciśnięcie Project -> Konfiguracja. Konfiguracja jest zapisana w formacji json. Przykładowo gdy projekt będzie miał nazwę pierwszyprojekt to plik będzie się nazywał pierwszyprojekt.json.

Przykładowy początkowy plik .json dla projektu pierwszyprojekt. 

  {
      "files": [
          {
              "fileLocation": 0,
              "filePath": "src/main.asm"
          }
      ],
      "memorySize": 1024,
      "name": "pierwszyprojekt",
      "stackPointer": 128,
      "startAt": 0,
      "version": "1.0.0"
  }

Struktura pliku

Pole "name" zawiera nazwę projektu, pole "memorySize" wielkość pamięci symulatora, "startAt" zawiera adres, od którego wykonywanie programu ma być rozpoczęte, "stackPointer" to adres stosu. Tablica "files" zawiera pliki i ich lokalizacje w pamięci w których mają być ustawione. Pole "version" to wersja pliku konfiguracyjnego, nie powinno być ręcznie zmieniane.

Szukaj/Zamień w projekcie

Projekty posiadają specjalną kartę do szukania i zamieniania ciągów znaków. Aby ją otworzyć należy wybrać menu Projekt -> Szukaj/Zamień. Pojawi się nowa karta nazwana nazwą projektu. W dolnej części karty są dwa pola oznaczone Znajdź i Zamień. W polu znajdź należy umieścić ciąg znaków którego chce się znaleść, a w polu zastąp ciąg znaków którm chce się go zastąpić. Poniżej pól są trzy przyciski oznaczone jako Szukaj, Zamień wybrane i Zamień wszystkie.

Po naciśnięciu przycisku szukaj projekt zostanie przeszukany i pliki które zawierają ten ciąg jak również lokalizacje tego ciągu w tych plikach zostaną wyświetlone w centralnej części karty. Znalezione wyniki można zaznaczyć klikając na nie lewym przyciskiem myszy. Po naciśnięciu przycisku Zamień wybrane ciągi znaków w wybranych plikach zostaną zamienione. Przycisk Zamień wszystkie zamienia wszystkie wystąpienia ciągu znaków.
