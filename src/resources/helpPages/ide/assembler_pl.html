Asembler

Środowisko posiada wbudowany asembler. Aby zasemblować napisany program należy przycisnąc przycisk Asemblacja. Jeżeli podczas procesu asemblacji wystąpiły jakieś błędy można je zobaczyć w sekcji Diagnostyka.

Jeżeli projekt jest otwarty to program będzie asemblowany według konfiguracji projektu, a gdy projekt nie jest otwarty to asemblowany będzie obecnie otwarty plik.
