JPE

Jump if parity even


If Parity = 1 execution will continue on specified memory address.

Instruction Hex code Length in bytes Length in cycles
JPE a16 0xEA 3 10

*a16 means 16 bits of address.

Example

MVI
B,3F
label: INR B
JPE
label
HLT

Becouse register B after incrementation will contain 0x40 and flag Parity will be 1 execution will continue on address of label.
