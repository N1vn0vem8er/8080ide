JNC

Jump if not carry


If Carry = 0 execution will continue on specified memory address.

Instruction Hex code Length in bytes Length in cycles
JNC a16 0xD2 3 10

*a16 means 16 bits of address.

Example

MVI
B,F2
MVI
A,10
SUB
B
JNC
label
HLT

Program will not jumt to address of label, because 0x10 - 0xF2 and Carry will be set to 1.
