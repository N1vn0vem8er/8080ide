JZ

Jump if zero


If Zero = 1 execution will continue on specified memory address.

Instruction Hex code Length in bytes Length in cycles
JZ a16 0xCA 3 10

*a16 means 16 bits of address.

Example

MVI
B,3F
label: INR B
JZ
label
HLT

Because register B after incrementation will contain 0x40 and flag Zero = 0 jump will not occur.
