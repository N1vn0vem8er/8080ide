CPI

Compare immediate with accumulator


Comperes accumulator and byte of immediate data after instruction code.

Contition bits affected: Carry, Sign, Zero, Parity, Auxiliary Carry

Instruction Hex code Length in bytes Length in cycles
CPI d8 0xFC 2 7

*d8 means 8 bits of data.

Example

MVI
A,05
CPI
03

After executing these instructions bit Zero will be set to 0, becaues values are not equal, Carry will be set to 0 informing, that contenst of accumulator are greater.
