RP

Return if plus


If Sign = 0 return will be performed.

Instruction Hex code Length in bytes Length in cycles
RP 0xF0 1 11/5

*length depends on action.

Example

MVI
A,F2
MVI
B,52
CALL
label
HLT
label: BSUB B
RP

After executing CALL instruction SUB B will be performed. Because 0xF2 - 0x52 = 0xA0 flag Sign will be set to 1 and return will not be performed.
