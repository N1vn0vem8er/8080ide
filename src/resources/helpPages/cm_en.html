CM

Call if minus


If Sign = 1 subroutine is called.

Instruction Hex code Length in bytes Length in cycles
CM a16 0xFC 3 17/11

*a16 means 16 bits of adderss.

*lenght depends on action.

Example

MVI
B,05
MVI
A,10
SUB
B
CM
label
HLT

Becouse 0x10 - 0x05 = 0x0B flag Sign = 0 and subroutine will not be called.
