CP

Call if plus


If Sign = 0 subroutine will be called.

Instruction Hex code Length in bytes Length in cycles
CP a16 0xE4 3 17/11

*a16 means 16 bits of address.

*length depends on action.

Example

MVI
B,05
MVI
A,10
SUB
B
CP
label
HLT

Becouse 0x10 - 0x05 = 0x0B flag Sign will be set to 0 and subroutine will be called.
