RPO

Return if parity odd


Jeżeli flaga Parity = 0 to operacja powrotu jest wykonywana

Instrukcja Kod hex Długość w bajtach Długość w cyklach
RPO 0xE0 1 11/5

*długość zależy od wykonania lub nie wykonania akcji.

Przykład

MVI
A,F2
MVI
B,52
CALL
label
HLT
label: SUB B
RPO

Po wykonaniu instrukcji CALL wykona się instrukcja SUB B. Ponieważ 0xF2 - 0x52 = 0xA0, 0xA0 % 2 == 0 flaga Parity zostanie ustawiona na 1 i operacja powrotu nie zostanie wykonana. Następnie wykona się instrukcja w następnym adresie pamięci po instrukcji RPO.
