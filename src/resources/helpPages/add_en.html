ADD

ADD register of memory to accumulator


Adds to contents of accumulator the contents of register or memory.

Condition bits affected: Carry, Sign, Zero, Parity, Auxiliary Carry

Instruction Hex code Lenght in bytes Length in cycles
ADD B 0x80 1 4
ADD C 0x81 1 4
ADD D 0x82 1 4
ADD E 0x83 1 4
ADD H 0x84 1 4
ADD L 0x85 1 4
ADD M 0x86 1 7
ADD A 0x87 1 4

Example

MVI
A,F2
MVI
B,04
ADD
B

After executing these instructions the content of the accumulator will be 0xF6.
