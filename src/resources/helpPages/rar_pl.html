RAR

Rotate accumulator right through carry


Zawartość akumulatora jest przesuwana o jedną pozycje w prawo. Bit z flagi Carry zapisywany jest na najmniej znaczącej pozycji, a bit z najbardziej znaczącej pozycji zapisywany jest w fladze Carry

Instrukcja Kod hex Długość w bajtach Długość w cyklach
RAR 0x1F 1 4

Przykład

MVI
A,6A
RRC

W fladze Carry jest 1. Po wykonaniu tych instrukcji zawartośc rejestru A będzie 0xB5 i flaga Carry zostanie ustawiona na 0.

Przed wykonaniem

Carry = 1
0 1 1 0 1 0 1 0

Po wykonaniu

1 0 1 1 0 1 0 1
Carry = 0
