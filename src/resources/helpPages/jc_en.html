JC

Jump if carry


If Carry = 1 execution will continue on specified memory address.

Instruction Hex code Length in bytes Length in cycles
JC a16 0xDA 3 10

*a16 means 16 bits of address.

Example

MVI
B,F2
MVI
A,10
SUB
B
JNC
label
HLT

Execution will continue on address of label, because 0x10 - 0xF2 Carry will be set.
