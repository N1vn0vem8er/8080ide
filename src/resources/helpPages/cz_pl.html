CZ

Call if minus


Warunkowa instrukcja wywołania podprogramu. Jeżeli flaga Zero = 1 program przechodzi do adresu podprogramu.

Instrukcja Kod hex Długość w bajtach Długość w cyklach
CZ a16 0xCC 3 17/11

*a16 oznacza 16 bitów adresu.

*długość zależy od wykonania lub nie wykonania akcji.

Przykład

MVI
B,05
MVI
A,10
SUB
B
CZ
label
HLT

Ponieważ 0x10 - 0x05 = 0x0B flaga Zero = 0 i podprogram nie zostanie wykonany.
