POP

Pop data off stack


The values ​​indicated by the SP stack pointer are loaded into the pair of registers. Data from the memory address pointed to by SP is loaded into the second register of the pair, and date at an address one greater than SP is loaded into the first register of the pair. If the PSW register pair is used, the data at address SP plus one is used to set the value of the flag register, at address SP the contents of the accumulator. Finally, the value in SP is increased by 2.

If POP PSW is used condition bits affected: Carry, Zero, Sign, Parity, Auxiliary Carry. Otherwise none bits affected.

Instruction Hex code Length in bytes Length in cycles Description
POP B 0xC1 1 10 Data is loaded to registers B and C.
POP D 0xD1 1 10 Data is loaded to registers D and E.
POP H 0xE1 1 10 Data is loaded to registers H and L.
POP PSW 0xF1 1 10 Data is loaded to flags register and accumulator.
