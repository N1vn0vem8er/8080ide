RPE

Return if parity even


If Parity = 1 return will be performed.

Instruction Hex code Length in bytes Length in cycles
RPE 0xE8 1 11/5

*length depends on action.

Example

MVI
A,F2
MVI
B,52
CALL
label
HLT
label: BSUB B
RPE

After executing CALL instruction SUB B will be performed. Because 0xF2 - 0x52 = 0xA0 flag Parity will be set to 1 and return operation will be performed. Next instruction that will be executed is HLT.
