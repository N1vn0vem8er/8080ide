JPO

Jump if parity odd


If Parity = 0 execution will continue on specified memory address.

Instruction Hex code Length in bytes Length in cycles
JPO a16 0xE2 3 10

*a16 means 16 bits of address.

Example

MVI
B,F1
label: INR B
JPO
label
HLT

Program will jump to label 1 time. Rregister B will be incremented to 0xF2 and execution will continue at address of label. Then B will be incremented to 0xF3 and jump will not occur.
