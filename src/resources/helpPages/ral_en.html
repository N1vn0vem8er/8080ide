RAL

Rotate accumulator left through carry


Contents of accumulator are rotated one bit position left. Most important bit is saved in Carry and Carry bit from carry is saved on least important bit.

Instruction Hex code Length in bytes Length in cycles
RAL 0x17 1 4

Example

MVI
A,B5
RAL

Carry is set to 0. After executing these instructions register A will contain 0x6A and flag Carry will be set to 1.

Before

Carry = 0
1 0 1 1 0 1 0 1

After

0 1 1 0 1 0 1 0
Carry = 1
