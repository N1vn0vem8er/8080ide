RRC

Rotate accumulator right


Contents of accumulator are rotated one bit left. Most important bit sets Carry.

Instruction Hex code Length in bytes Length in cycles
RRC 0x0F 1 4

Example

MVI
A,F2
RRC

After executing these instruction register A will contain 0x79 and Carry will be set to 0.

Before

Carry = x
1 1 1 1 0 0 1 0

After

0 1 1 1 1 0 0 1
Carry = 0
