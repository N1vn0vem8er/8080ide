JNZ

Jump if not zero


If Zero = 0 execution will continue on specified memory address.

Instruction Hex code Length in bytes Length in cycles
JNZ a16 0xC2 3 10

*a16 means 16 bits of address.

Example

MVI
B,03
label: DCR B
JNZ
label
HLT

Program will jump to address of label as long as register B will not contain 0.
