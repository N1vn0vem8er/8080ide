JP

Jump if positive


If Sign = 0 execution will continue on specified memory address.

Instruction Hex code Length in bytes Length in cycles
JP a16 0xF2 3 10

*a16 means 16 bits of address.

Example

MVI
B,3F
label: INR B
JP
label
HLT

Because register B after incrementation will contain 0x40 and flag Sign will be 0 execution will continue on address of label.
