CPE

Call if parity even


If Parity = 1 subroutine will be called.

Instruction Hex code Length in bytes Length in cycles
CPE a16 0xEC 3 17/11

*a16 means 16 bits of address.

*length depends on action.

Example

MVI
B,05
MVI
A,10
SUB
B
CPE
label
HLT

Becouse 0x10 - 0x05 = 0x0B flag Parity will be set to 0 and subroutine will not be called.
