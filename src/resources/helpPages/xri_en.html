XRI

Exclusive-Or immediate with accumulator


Logical XOR on accumulator and immediate data. Carry bit is always set to 0.

Condition bits affected: Carry, Sign, Zero, Parity

Instruction Hex code Length in bytes Length in cycles
XRI d8 0xEE 2 7

*d8 means 8 bits of data.

Example

MVI
A,05
XRI
03

After executing these instructions accumulator will contain 0x06.
