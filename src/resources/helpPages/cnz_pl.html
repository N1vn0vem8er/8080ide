CNZ

Call if not zero


Jeżeli flaga Zero = 0 program przechodzi do podprogramu.

Instrukcja Kod hex Długość w bajtach Długość w cyklach
CNZ a16 0xC4 3 17/11

*a16 oznacza 16 bitów adresu.

*długość zależy od wykonania lub nie wykonania akcji.

Przykład

MVI
B,05
MVI
A,10
SUB
B
CNZ
label
HLT

Ponieważ 0x10 - 0x05 nie jest równe 0 flaga Zero zostanie ustawiona na 0 i podprogramu się wykona.
