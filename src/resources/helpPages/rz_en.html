RZ

Return if zero


If Zero = 1 return is performed

Instruction Hex code Length in bytes Length in cycles
RZ 0xC8 1 11/5

*length depends on action.

Example

MVI
A,F2
MVI
B,52
CALL
label
HLT
label: BSUB B
RZ

After executing CALL instruction SUB B will be performed. Because 0xF2 - 0x52 = 0xA0 flag Zero will be set to 0 and return will not be performed.
