RNC

Return if not carry


Jeżeli flaga Carry = 0 to operacja powrotu jest wykonywana

Instrukcja Kod hex Długość w bajtach Długość w cyklach
RNC 0xD0 1 11/5

*długość zależy od wykonania lub nie wykonania akcji.

Przykład

MVI
A,F2
MVI
B,52
CALL
label
HLT
label: BSUB B
RNC

Po wykonaniu instrukcji CALL wykona się instrukcja SUB B. Ponieważ 0xF2 > 0x52 flaga Carry zostanie ustawiona na 0 i instrukcja RNC sprawi, że program powróci do adresu instrukcjiCALL. Następnie wykona się instrukcja HLT.
