RNZ

Return if not zero


Jeżeli flaga Zero = 0 to operacja powrotu jest wykonywana

Instrukcja Kod hex Długość w bajtach Długość w cyklach
RNZ 0xC0 1 11/5

*długość zależy od wykonania lub nie wykonania akcji.

Przykład

MVI
A,F2
MVI
B,52
CALL
label
HLT
label: BSUB B
RNZ

Po wykonaniu instrukcji CALL wykona się instrukcja SUB B. Ponieważ 0xF2 - 0x52 != 0 flaga Zero zostanie ustawiona na 0 i instrukcja RNZ sprawi, że program powróci do adresu instrukcjiCALL. Następnie wykona się instrukcja HLT.
