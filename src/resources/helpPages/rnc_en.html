RNC

Return if not carry


If Carry = 0 return is performed.

Instruction Hex code Length in bytes Length in cycles
RNC 0xD0 1 11/5

*length depends on action.

Example

MVI
A,F2
MVI
B,52
CALL
label
HLT
label: BSUB B
RNC

After executing CALL instruction SUB B will be performed. Because 0xF2 > 0x52 flag Carry will be set to 0 and instruction RNC will perform return operation. Next instruction that will be executed is HLT.
