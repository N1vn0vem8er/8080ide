STAX

Store accumulator


Stores contents of accumulator in memory. Memory address is pointed by register pair written in instruction.

Instruction Hex code Length in bytes Length in cycles Description
STAX B 0x02 1 7 Memory address is pointed by B and C.
STAX D 0x12 1 7 Memory address is pointed by D and E.

Example

MVI
A,AB
MVI
B,00
MVI
C,10
STAX
B

After executing these instructions contents of register A that is 0xAB will be saved under memory address 0x0010.
