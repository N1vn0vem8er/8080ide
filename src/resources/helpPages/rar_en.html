RAR

Rotate accumulator right through carry


Contents of accumulator are rotated one bit position right. Most important bit is saved in Carry and Carry bit from carry is saved on least important bit.

Instruction Hex code Length in bytes Length in cycles
RAR 0x1F 1 4

Example

MVI
A,6A
RRC

If carry contains 1. After executing these Instruction register A will contain 0xB5 and Carry will be set to 0.

Before

Carry = 1
0 1 1 0 1 0 1 0

After

1 0 1 1 0 1 0 1
Carry = 0
