Graphical Screen

The graphical screen allows for the display of 256x256 pixel graphics in 256 colors. To open the screen, select Simulator -> Screen from the menu. The screen is operated using four microprocessor outputs. These outputs are as follows:

Output Action
02 Set cursor x-position
03 Set cursor y-position
04 Apply specified color
05 Select command

Drawing on the screen involves selecting the cursor position using outputs 2 and 3, and then providing the color you wish to use on output 4. Colors are stored in RGB 3-3-2 format. The first three bits represent red, the next three bits represent blue, and the last two represent green. All available colors can be viewed by opening Help -> Screen Colors Palette. Normally, sending a color to output 4 will color the pixel where the cursor is currently set. The screen also features a series of commands that allow for easy coloring of entire shapes. Using most commands involves selecting the shape's coordinates, the shape is drawn the moment the color is provided.

Command Action
0 Clear screen
01 Set the first position for line generation at the cursor location.
02 Set the second position for line generation at the cursor location.
03 Sets the next position at the cursor location for area filling (repeatable).
04 Sets the next position at the cursor location for drawing an arbitrary polygon (repeatable).
05 Fill the entire screen with the specified color.
06 Mark the center of a filled ellipse at the cursor location.
07 Mark the radius of a filled ellipse at the cursor location.
08 Mark the center of an unfilled ellipse at the cursor location.
09 Mark the radius of an unfilled ellipse at the cursor location.
FF Cancel command.
