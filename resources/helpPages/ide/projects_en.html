Projects


Projects allow you to create programs from multiple files, use integration with the git version control system, as well as set up the simulator. They are the preferred way to create medium to larger programs.

Creating project

To create a project, press the Project menu -> New Project. A window will appear asking you to enter the location of the project, its name and the initially set memory size of the simulator. After pressing the Ok button, the project will be created. The Git repository will also be initialized. If there are any errors in the project configuration, it will not be created.

Project configuration

The project is configured by editing the {project name}.json file or by pressing Project menu -> Configure. For example, if the project will be named exampleproject, the file will be called exampleproject.json.

Sample initial .json file for project exampleproject. 

{
    "files": [
        {
            "fileLocation": 0,
            "filePath": "src/main.asm"
        }
    ],
    "memorySize": 1024,
    "name": "pierwszyprojekt",
    "stackPointer": 128,
    "startAt": 0,
    "version": "1.0.0"
}

File structure

The "name" field contains the project name, the "memorySize" field the simulator's memory size, "startAt" the address from which program execution should begin, and "stackPointer" the stack address. The "files" array contains files and their memory locations where they should be set. The "version" field is the version of the configuration file and should not be manually changed.

Search/Replace in project

Projects have a special tab for searching and replacing strings of characters. To open it, select the Project -> Search/Replace. A new tab will appear named after your project. At the bottom of the tab there are two boxes labeled Search and Replace. In the find field, enter the string of characters you want to find, and in the replace field, the string of characters you want to replace it with. Below the fields are three buttons labeled Search, Replace Selected, and Replace All.

After pressing the search button, the project will be searched and the files that contain this string as well as the locations of this string in these files will be displayed in the central part of the tab. The found results can be marked by clicking on them with the left mouse button. After pressing the Replace button, selected strings of characters in the selected files will be replaced. The Replace All button replaces all occurrences of a string.
